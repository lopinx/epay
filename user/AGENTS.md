# 用户中心

## 概述
用户中心是商户操作界面，提供商户注册、登录、支付配置、订单管理等功能。

## 目录结构
```
user/
├── index.php       # 用户中心入口
├── login.php       # 登录页面
├── reg.php         # 注册页面
├── findpwd.php     # 找回密码
├── oauth.php       # 第三方登录
├── test.php        # 测试支付
├── cert.php        # 实名认证
├── apply.php       # 商户申请
├── work.php        # 工单系统
├── agreement.php   # 服务协议
├── invite.php      # 邀请返现
├── assets/         # 静态资源文件
│   ├── css/
│   ├── js/
│   └── img/
└── pay/            # 支付相关页面
    ├── alipay.php
    ├── wxpay.php
    ├── qqpay.php
    └── ...
```

## 关键功能
1. 商户注册和登录
2. 支付接口配置
3. 订单和结算管理
4. 资金记录查看
5. 实名认证申请
6. 测试支付功能
7. 邀请返现管理

## 特殊约定
1. 使用独立的商户验证机制
2. 商户只能查看和操作自己的数据
3. 支付测试功能仅供商户调试使用
