# 模板系统

## 概述
模板系统负责前端页面展示，包含多个不同风格的模板目录，支持前台展示和支付页面。

## 目录结构
```
template/
├── default/        # 默认模板
├── index1/         # 模板1
├── index2/         # 模板2
├── index3/         # 模板3
├── index4/         # 模板4
├── index5/         # 模板5
├── index6/         # 模板6
├── index7/         # 模板7
├── index8/         # 模板8
├── index9/         # 模板9
├── index10/        # 模板10
└── pay/            # 支付页面模板
```

## 模板结构
每个模板目录通常包含：
1. `index.php` - 模板首页文件
2. `assets/` - 静态资源文件
3. `user/` - 用户中心模板
4. `admin/` - 管理后台模板
5. `doc/` - 文档页面模板

## 模板切换
1. 系统支持多套模板自由切换
2. 模板通过后台进行设置
3. 不同页面可以使用不同模板

## 特殊约定
1. 模板文件使用PHP语法
2. 模板变量通过全局数组传递
3. 模板继承和包含机制
